# portfolio-git

Repositório pessoal de aprendizagem em Git e controle de versão, desenvolvido como parte da disciplina CIN0136 — Desenvolvimento de Software (CIn-UFPE, 2026.1).

## Índice

- [exercicios/ex01-commits](./exercicios/ex01-commits/) — prática de commits semânticos
- [exercicios/ex02-branches](./exercicios/ex02-branches/) — prática de branches e merges
- [exercicios/ex03-conflito](./exercicios/ex03-conflito/) — resolução de merge conflict
- [projeto/hello-world](./projeto/hello-world/) — mini-projeto usado nos exercícios

## Como usar

Clone o repositório e explore as pastas de exercícios em ordem. Cada uma documenta uma etapa diferente do aprendizado com Git.

```bash
git clone https://github.com/bernardobelfort/portfolio-git.git
cd portfolio-git
```

## Tecnologias

- Git
- GitHub CLI (`gh`)
- JavaScript (Node.js)

## Sobre o exercício

Este repositório foi construído como parte do EI1 — Portfólio Git da disciplina CIN0136. O objetivo foi praticar commits semânticos, branches, Pull Requests e resolução de conflitos de forma estruturada.
